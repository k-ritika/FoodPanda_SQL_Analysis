# 🍽️ Food Panda SQL Analysis

This project explores customer ordering behavior on Food Panda using SQL queries and SQLite. It focuses on segmentation, churn prediction, and pricing trends — all executed through DBeaver and organized for reproducible analytics.

---

## 📁 Project Structure

| File | Description |
|------|-------------|
| `Script-5.sql` | Contains all SQL queries used for analysis |
| `foodpanda_analysis.db` | SQLite database file with customer and order data |
| `README.md` | Project overview and usage instructions |

---

## 🔍 Key Insights

- **Customer Segmentation**: Grouped by city, age, and gender
- **Order Frequency**: Identified loyal vs. infrequent customers
- **Churn Prediction**: Flagged users with long gaps since last order
- **Price Trends**: Analyzed dish categories and average spend
